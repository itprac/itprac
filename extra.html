Basic Commands:

1) To search a string in a file:
grep "string" filename

2) To redirect the output of any command to a file:
E.g. to redirect output of 'ls' command to a file 'f1'
ls > f1

3) To archive files of any directory
E.g. to archive files of bin directory
tar -cvf bin.tar /bin

4) To extract files from archive
E.g. to extract files of bin.tar
tar -xvf bin.tar

5) To show all files that start with character 'f'
ls f*

6) To show all files that ends with character 'f'
ls *f

7) To change group ownership of file 'f1' to group 'grp1'
sudo chgrp grp1 f1



🧩 1. Basic Commands of Linux

Here are some of the most commonly used Linux basic commands:

Command Description
pwd Shows the current working directory
ls Lists files and folders in the directory
cd Changes the directory
mkdir Creates a new directory
rmdir Removes an empty directory
cp source destination Copies files or directories
mv source destination Moves or renames files/directories
rm filename Deletes a file
cat filename Displays file content
touch filename Creates an empty file
chmod Changes file permissions
chown Changes file owner
clear Clears the terminal screen
man command Shows manual/help for a command
📦 2. How to Archive a File

You can archive (compress) files using commands like tar or gzip.

👉 Using tar (most common)

To create an archive:

tar -cvf archive_name.tar file1 file2 file3


c → create archive

v → verbose (shows progress)

f → filename for the archive

Example:

tar -cvf myarchive.tar myfile.txt myfolder/

👉 To extract an archive
tar -xvf archive_name.tar

👉 To create a compressed archive
tar -czvf archive_name.tar.gz files/


(z adds gzip compression)

💻 3. Basic Syntax for Shell Scripting in Linux

A shell script is a file containing commands that are executed by the shell (like bash).

🧠 Basic Syntax:

Start with the Shebang line:

#!/bin/bash


This tells the system that the file should be run in the Bash shell.

Write commands line by line:
Example:

#!/bin/bash
echo "Hello, World!"
pwd
ls


Make the script executable:

chmod +x scriptname.sh


Run the script:

./scriptname.sh

Example: Simple Script with Variables and Conditions
#!/bin/bash
name="Aimbot"
echo "Hello $name"

if [ $name == "Aimbot" ]; then
echo "Welcome to Linux practice!"
else
echo "Unknown user"
fi
